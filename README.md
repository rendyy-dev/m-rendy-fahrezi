## 🎨 Theme: Dark Emerald Neon
Website ini mengusung tema **Dark Mode** dengan aksen **Emerald Green Neon** untuk memberikan kesan modern, tech-savvy, dan profesional. Menggunakan pendekatan *Clean & Minimalist Design* untuk memastikan pengalaman pengguna (UX) yang maksimal.

## 🛠️ Tech Stack
Website ini dibangun dengan teknologi modern untuk performa optimal:
- **Core:** HTML5, CSS3, JavaScript (ES6+)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Utility-first framework)
- **Icons:** FontAwesome / Lucide Icons
- **Deployment:** [Vercel](https://vercel.com/)
- **Version Control:** GitHub

## 📂 Project Structure
Saya menerapkan struktur file yang terorganisir untuk memudahkan *maintenance*:
- `index.html` - Main Landing Page
- `about.html` - Detail Profil & Journey
- `portofolio.html` - Showcase Project (Case Studies)
- `services.html` - Layanan & Penawaran Bisnis
- `contact.html` - Integrasi Kontak & WhatsApp
- `components/` - Modul Reusable (Navbar & Footer)
- `js/components.js` - Logic untuk Dynamic Component Injection
