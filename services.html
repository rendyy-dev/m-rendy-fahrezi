Pertanyaan Umum

Berapa lama waktu pengerjaan?

Landing page 1 halaman: 3-5 hari. Multi-halaman: 5-7 hari. Web dinamis: 2-4 minggu tergantung kompleksitas.

Apakah termasuk hosting?

Untuk paket Web Dinamis termasuk setup hosting. Untuk paket landing page, bisa dibantu setup di hosting Anda atau direkomendasikan hosting terbaik.

Bagaimana sistem pembayaran?

Sistem bertahap: 50% DP, 50% setelah proyek selesai dan disetujui. Untuk paket tertentu bisa dinegosiasikan.

Apakah source code diberikan?

Ya, source code lengkap diberikan setelah proyek selesai. Untuk web dinamis termasuk dokumentasi dan panduan penggunaan.
